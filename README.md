Exploring pandas for data manipulation and analysis as part of expanding 
my backend skill set toward data-intensive and AI/ML applications.

## Topics Covered
- DataFrame operations (read_csv, read_excel, read_json)
- EDA basics — info(), describe(), head(), tail()
- shape , column, series, dataframe
- searching and filtering
- Data cleaning and transformation
- Modification of columns, Removing 
- Handling Missing Data , Values
- Interpolation
- Summary
- Grouping, Multi Grouping
- Manipulate Data
- Sorting and Aggregation
- Merging and Filtering (joins)
- Concanting DataFrame

## Tech
- Python 3.13
